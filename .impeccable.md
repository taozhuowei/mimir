# Scales Tarot — Design Context

## Design Context

### Users
轻娱乐用户：对塔罗有好奇心，在面对两难决策时希望借助一种有仪式感的方式帮助思考。不是专业占卜师，但希望体验是认真的、有沉浸感的——而不是游戏化的。使用场景通常是个人独处，带着一定的情绪投入。

### Brand Personality
**神秘 · 仪式 · 权威**

- 像一位古老的神谕，沉默而精准，不解释、不讨好
- 每一步操作都应有仪式感，不是"点击按钮"，而是"召唤力量"
- 结果呈现要有分量——不是轻飘飘的推送，而是郑重的揭示

### Aesthetic Direction
**方向**：从现有暖色羊皮纸风格向深色神秘感偏移，保留金属质感但加深整体色调。

**参考感受**：古欧式占星图、魔法阵、深夜礼拜堂、中世纪手稿的插图边框

**色调**：
- 背景 → 深棕近黑 (`#1A0F0A`)，带隐约的暗紫/墨蓝
- 主调 → 深暗棕紫 (`#2D1B14`、`#3D2518`)
- 金属 → 暗金 / 旧铜，非纯亮金（`#C9973A`、`#8B6F2E`、带做旧纹理）
- 文字 → 羊皮纸米色 (`#E8D4A8`) 在深背景上，保持古典质感

**字体**：保留 Cinzel（展示）+ Cormorant Garamond（正文），适合神秘古典调性

**装饰元素**：
- SVG 塔罗主题进度图标（三阶段图标不同：月亮 / 五芒星 / 太阳）
- 使用 SVG 线性/径向渐变实现金属质感
- 星图纹理、符文边框、神秘光晕

**反参考**（不要做成）：
- 过于鲜亮的紫色"算命 App"风
- 游戏化、卡通化
- 极简白底科技感

### Design Principles

1. **仪式优先于效率**：每一个交互节点都是仪式的一部分——不用清楚解释操作，让氛围引导用户
2. **克制的戏剧性**：动效和装饰要有冲击力，但不能喧宾夺主；金属质感要精致，不能俗气
3. **深色即深度**：暗色背景不是"暗黑系"，而是营造夜深一人独对的私密感
4. **文字有分量**：字间距、大写、衬线字体——每一行文字都像刻在石头上的
5. **不过度解释**：界面只说必要的话，结果揭示是主角，操作提示是配角

## 跨端兼容原则（H5 & 微信小程序）

**目标**：H5 端和微信小程序端的视觉表现与交互体验必须完全一致。

### 强制约束（小程序不支持的 API）

| 禁止使用 | 替代方案 |
| --- | --- |
| `window.innerWidth / innerHeight` | `uni.getWindowInfo().windowWidth / windowHeight` |
| `window.addEventListener('resize', fn)` | `uni.onWindowResize(fn)` |
| `window.removeEventListener('resize', fn)` | `uni.offWindowResize(fn)` |
| `el.getBoundingClientRect()` | 从 `uni.getWindowInfo()` 推算，或 `uni.createSelectorQuery()` |
| `el.offsetWidth / offsetHeight` | 从 CSS 变量 + 窗口尺寸推算 |
| `document.querySelector / getElementById` | UniApp 模板 ref 或 `uni.createSelectorQuery()` |
| `GSAP.to(domElement, ...)` | plain JS 状态对象 + `onUpdate` → Vue `ref<string>` `:style` 绑定 |

### GSAP 跨端动画模式

微信小程序没有 DOM，GSAP 不能直接操作元素。标准模式：

```text
plain JS 状态对象（_bg, _initials[], ...）
    ↓ GSAP tween（onUpdate 回调）
刷新函数（refreshXxx）
    ↓ 修改 Vue ref<string>
Vue :style 绑定
    ↓ Vue 渲染
最终 DOM/视图更新
```

- `centerStyle` 函数用 `calc(-50% + Xpx)` 代替 GSAP 的 `xPercent/yPercent: -50`
- 元素显隐用 `v-show`，不用 GSAP 的 `autoAlpha` / `display`

### CSS 跨端约束

| 禁止（WXSS 不支持） | 替代 |
| --- | --- |
| `* { }` 通配选择器 | 用 `/* #ifdef H5 */` 包裹 |
| `html { }` 选择器 | 同上，小程序用 `page { }` |
| `@import url(...)` Google Fonts | 本地 `@font-face` + `/* #ifdef H5 */` |
| `.parent > *` 通配子选择器 | 枚举子元素类型，如 `.parent > view, .parent > image` |
