# Scales Tarot 产品需求

Scales Tarot 是一个以 H5 为主要交付形态的单页塔罗体验产品。用户进入首页后，在同一页面内完成牌阵选择、仪式化抽牌、结果揭示与文字解读，全程无需跳转。产品以最短路径、稳定反馈与一致视觉氛围，提供一次完整、可信、可重复的占卜体验。

当前正式范围为 H5 与单张牌阵（`single_card`），不含账号、历史记录、支付、社交分享、多主题切换与小程序发布。

核心链路：待机视图（摊牌动画引导）→ 点击触发 → 占卜视图（洗牌 / 切牌 / 抽牌 / 翻牌仪式动画）→ 结果卡牌定格 → 解读视图（宽屏分栏 / 窄屏抽屉，打字机呈现结论与逐张牌义）→ 决策（再占一次 / 回到首页 / 异常重试）。

详细需求按模块拆分于 [prd/](prd/)：

- [prd/state.md](prd/state.md) — 状态机：流程阶段、占卜相位、视图与阶段对应、用户流程、异常恢复
- [prd/view.md](prd/view.md) — 视图与视觉层、视图相关功能需求、交互原则
- [prd/animation.md](prd/animation.md) — 动画分帧、视图过渡动画、动效规范
- [prd/glossary.md](prd/glossary.md) — 项目术语表
